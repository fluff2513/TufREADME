# TufREADME 🌈

> Too Useful For *README* - v1

A simple interactive Python script that generates a README.md file for your project.

## 🌴 Installation

```
git clone https://github.com/Fluff2513/TufREADME
cd TufREADME
pip install -r requirements.txt
python tuffrm.py
```

## 🎀 Usage

```
python tuffrm.py
```

Then just answer the questions - TufMD does the rest!
